Bienvenido a la Microcredencial en Accesibilidad Digital

Este programa de especialización está diseñado para profesionales que desean convertirse en expertos en accesibilidad web y digital, garantizando que todos los usuarios, independientemente de sus capacidades, puedan acceder y utilizar contenidos digitales de manera efectiva.

Características de Accesibilidad Implementadas

  • Navegación por teclado: Todos los elementos son accesibles mediante Tab y Enter
  • ARIA labels: Etiquetas descriptivas para lectores de pantalla
  • Contraste adecuado: Cumple con WCAG 2.1 nivel AA
  • Skip link: Enlace para saltar al contenido principal
  • Responsive design: Adaptable a todos los tamaños de pantalla
  • Indicadores de foco: Elementos claramente visibles al navegar por teclado
  • HTML semántico: Estructura clara con header, nav, main y roles ARIA
  • Estados del menú: aria-expanded comunica el estado del menú móvil
